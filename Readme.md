# Go -selpg



---

使用Go实现Linux命令行程序selpg
## 功能简介
在输入（stdin或文件）的内容中，根据指定-s [Number] -e [Number]参数，选择指定页码范围内的内容输出，同时可以通过-d [destPrinter]将输出内容直接传送到目标打印机进行打印。
